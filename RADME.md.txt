Тестирование с Cypress
Описание проекта
Данный репозиторий содержит набор тестов UI-автоматизации, 
написанных с использованием инструмента Cypress.

Установка зависимостей
Для запуска тестов потребуется установить следующие зависимости:

npm install cypress --save-dev


Чтобы запустить локальные тесты в браузере Cypress:

npx cypress open

Для автоматического запуска всех тестов без открытия интерфейса:

npx cypress run
